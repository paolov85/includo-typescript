# IncluDO — Progetto TypeScript

Progetto del corso TypeScript (start2impact): un sistema che modella la struttura operativa di una scuola di formazione professionale per migranti, con interazioni tra partecipanti, corsi e aziende partner.

## Stato attuale

Lavoro in corso — definite le interfacce (`IPartecipante`, `ICorso`, `IAzienda`) e le classi che le implementano (`Partecipante`, `Corso`, `Azienda`) in `console/includo-console.ts`. Prossimi passi: istanziazione e test della logica, interfaccia interattiva.
