# IncluDO — Progetto TypeScript

Progetto del corso TypeScript (start2impact): un sistema che modella la struttura operativa di una scuola di formazione professionale per migranti, con interazioni tra partecipanti, corsi e aziende partner.

## Stato attuale

Lavoro in corso — definite le prime interfacce (`IPartecipante`, `ICorso`) in `console/includo-console.ts`. Prossimi passi: interfaccia `IAzienda`, classi che le implementano, test della logica, interfaccia interattiva.
