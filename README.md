# IncluDO — Progetto TypeScript

Progetto del corso TypeScript (start2impact): un sistema che modella la struttura operativa di una scuola di formazione professionale per migranti, con interazioni tra partecipanti, corsi e aziende partner.

## Stato attuale

Entrambe le versioni del progetto sono complete:
- **`console/includo-console.ts`** — interfacce, classi, dati di esempio e test della logica tramite `console.log`.
- **`interattivo/`** — stessa logica con interfaccia HTML interattiva (`index.html`, `styles.css`, `includo.ts`): form per aggiungere partecipanti/corsi/aziende, iscrizioni e offerte di lavoro in tempo reale, con log attività a schermo.

Prossimo passo: rifinitura finale (istruzioni d'uso complete nel README) e deploy su CodePen.
